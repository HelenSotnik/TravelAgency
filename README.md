# TravelAgency
Travel Agency SpringMvc + Hibernate Application wit Spring Security provides User or Manager with posibility to seach for list of hotels in intrested location and to book the room in hotel if it is available for chosen period. Manager can add hotels and rooms, view all users and their orders.
